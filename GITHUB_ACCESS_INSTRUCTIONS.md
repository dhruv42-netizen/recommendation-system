## 2. Uploading the Project to GitHub

### Step 1 — Create GitHub Repository
1. Go to [https://github.com/new](https://github.com/new)
2. Enter repository name: `recommendation-system`
3. Add description: `Recommendation system backend using Node.js and similarity algorithms`
4. Set visibility to **Private**
5. Do NOT check "Add a README file"
6. Click **Create repository**

### Step 2 — Upload Files
1. On the new repository page, click **uploading an existing file**
2. Drag and drop all project files:
   - `src/index.js`
   - `package.json`
   - `README.md`
   - `.gitignore`
3. Add commit message: `Initial commit: Recommendation System`
4. Click **Commit changes**

---

## 3. Granting Write Access to Trainer

### Step 1 — Open Repository Settings
1. Go to your repository on GitHub
2. Click the **Settings** tab
3. In the left sidebar, click **Collaborators and teams**

### Step 2 — Add Collaborator
1. Click **Add people**
2. Search for your trainer's GitHub username or email
3. Select their account

### Step 3 — Set Permission Level

| Role | Recommended? |
|---|---|
| Read | ❌ |
| Write | ✅ Yes — select this |
| Admin | ❌ Avoid |

### Step 4 — Send Invitation
1. Click **Add to repository**
2. Trainer receives an email — they must **accept the invitation**

## 5. Revoking Access After Review

1. Go to **Settings → Collaborators and teams**
2. Find trainer's username
3. Click **Remove**
